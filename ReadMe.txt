git 基本操作命令:

三个树： 
A.工作目录【实际看到的文件】；
B.暂存区(Index),临时保存改动；
C.HEAD 最后一次提交的结果。

D.推送改动 push ；

1.创建新仓库 git init
2.检出仓库 git clone/path/to/repository  远程 git clone username@host:/path/to/repository
3.添加 git add filename/*[all]
4.提交： git commit -m "注释信息"
5.本地仓连接 git remote add origin <server>
6.创建分支：git checkout -b fenzhiname
7.切换主分支：git checkout master
8.删除分支： git branch -d fenzhiname
9.推送分支到远程：git push origin <branch>
10.更新：git pull
11.合并其他分支到当前分支 git merge <barnch>
